# 🎉 Setup.exe V1.7.0 - Lightweight IPTV App for Windows

## ✨ Features

- 🌍 **10,000+ TV Channels Worldwide**: Access a vast selection of channels, including essential Indian channels.
- 🎮 **Keyboard Navigation**:
  - ⬆️⬇️ **Up/Down**: Adjust volume.
  - ⬅️➡️ **Left/Right**: Navigate categories and switch channels.
  - ⭐ **`F` Key**: Mark a channel as a favorite.
- ⚡ **Lightweight and Efficient**: Optimized for performance with regular updates.
- 📡 **Based on [iptv.org](https://iptv.org)**: Reliable and community-driven source.

## 🔒 Permissions Required

- 🔗 **Private Network Access**: Ensure seamless streaming within your network.
- 🛡️ **Run as Administrator**: Required for optimal functionality.

## 🖥️ Compatibility

- **Platform**: Windows.
- **Version**: Fully compatible with Windows 10 and above.

## 🛠️ Installation

1. 📥 Download the `Setup.exe` file.
2. 🚀 Run the installer with administrator privileges.
3. ✅ Grant private network access when prompted.

Enjoy a seamless IPTV experience with regular updates and support for your favorite channels! 🎬
